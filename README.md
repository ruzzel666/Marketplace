# 🛍️ Willberries Marketplace

Willberries — это современный, легкий и отзывчивый интернет-магазин одежды и аксессуаров, реализованный на чистом **JavaScript (Vanilla JS)**, **HTML5** и **CSS3** с использованием адаптивной сетки **Bootstrap Grid** и слайдера **Swiper**. В качестве бэкенда для хранения каталога товаров и заказов используется локальный REST API сервер на базе **json-server**.

---

## ✨ Ключевые возможности

*   **⚡ Динамический каталог товаров:** Фильтрация товаров по категориям (*Clothing*, *Shoes*, *Accessories*), полу (*Womens*, *Mens*) или просмотр всех товаров. Переход между главной страницей (`index.html`) и страницей каталога (`goods.html`) происходит бесшовно с сохранением выбранных фильтров.
*   **🔍 Полнотекстовый поиск:** Удобная поисковая строка в шапке сайта, которая фильтрует товары по названию и отображает результаты на странице каталога.
*   **🛒 Интерактивная корзина с сохранением состояния:**
    *   Добавление товаров в корзину с динамическим обновлением счетчика на кнопке в шапке сайта.
    *   Синхронизация и сохранение корзины в `localStorage` (корзина сохраняется при перезагрузке страницы или переходе между страницами).
    *   Модальное окно корзины с управлением количеством товаров: увеличение (`+`), уменьшение (`-`) и удаление (`x`) товаров прямо из списка.
    *   Автоматический перерасчет стоимости каждого пункта и общей суммы заказа (`Total`).
*   **📦 Умное оформление заказа (Checkout):**
    *   Форма в корзине для ввода контактных данных (Имя и Телефон).
    *   Отправка данных заказа на локальный сервер `http://localhost:3000/orders`.
    *   **Резервный механизм (Fallback):** В случае, если локальный сервер не запущен или недоступен, заказ автоматически отправляется на публичный тестовый эндпоинт `JSONPlaceholder` (`https://jsonplaceholder.typicode.com/posts`), благодаря чему интерфейс успешно завершает оформление заказа и очищает корзину без критических ошибок.
*   **🎠 Интерактивный промо-слайдер:** Плавный слайдер на главной странице, реализованный с помощью Swiper JS, позволяющий быстро добавлять акционные товары в корзину в один клик.

## 🛠️ Стек технологий

*   **Фронтенд:** HTML5, CSS3, JavaScript (ES6+ / Vanilla JS)
*   **Адаптивная сетка:** Bootstrap Grid (`bootstrap-grid.min.css`)
*   **Слайдер:** Swiper JS (`swiper-bundle.min.css`)
*   **Локальный сервер / БД:** `json-server` (эмуляция базы данных через файл [db.json](file:///c:/Users/gnz29/Documents/dev/2026/Nuxt/Marketplace/db/db.json))
*   **State Management:** `localStorage` (хранение корзины и фильтров каталога)

## 🚀 Быстрый старт

### 1. Установка зависимостей
Убедитесь, что у вас установлен Node.js. Установите зависимости проекта (включая локальный `json-server`):
```bash
npm install
```

### 2. Запуск локального API-сервера
Запустите `json-server` для обслуживания базы данных товаров и приема заказов:
```bash
npx json-server db/db.json
```
Сервер запустится по адресу `http://localhost:3000`. Будут доступны следующие эндпоинты:
*   Каталог товаров: `http://localhost:3000/db`
*   Список оформленных заказов: `http://localhost:3000/orders`

### 3. Запуск фронтенда
Поскольку проект разработан на чистом HTML, вы можете открыть его двумя способами:
1.  **Простой:** Дважды кликните по файлу [index.html](file:///c:/Users/gnz29/Documents/dev/2026/Nuxt/Marketplace/index.html), чтобы открыть его прямо в браузере.
2.  **Через локальный сервер статики (рекомендуется):** Запустите локальный сервер, например, с помощью расширения *Live Server* в VS Code или выполнив команду в консоли:
    ```bash
    npx serve .
    ```
    После этого перейдите по адресу, который выдаст утилита (обычно `http://localhost:3000` или `http://localhost:5000`).
