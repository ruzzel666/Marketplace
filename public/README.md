# 🛍️ Willberries Marketplace

Willberries — это современный, быстрый и отзывчивый интернет-магазин одежды и аксессуаров, разработанный на фреймворке **Nuxt 3** с интеграцией базы данных **Firebase Realtime Database**. Проект полностью оптимизирован для статической генерации (SSG) и развертывания на **GitHub Pages**.

---

## ✨ Ключевые возможности

*   **⚡ Мгновенный каталог и поиск:** Клиентская фильтрация товаров по категориям (Одежда, Обувь, Аксессуары), полу (Мужское/Женское) и текстовому поиску без лишних сетевых запросов.
*   **🎠 Интерактивный промо-слайдер:** Плавный слайдер на главной странице с аппаратным ускорением (`transform`), перелистывающий уникальные промо-изображения и информацию.
*   **🛒 Функциональная корзина:**
    *   Добавление товаров, регулирование количества и авторасчёт итоговой суммы.
    *   Динамический пустой экран («Корзина пуста») с автоматическим скрытием строки `Total`.
*   **🔥 Интеграция с Firebase:** Оформление заказа отправляет данные клиента и список товаров напрямую в Firebase Realtime Database.
*   **🚀 Готовность к GitHub Pages:** Проект настроен для деплоя на статический хостинг с обходом кэширования роутов и поддержкой подпутей.

---

## 🛠️ Стек технологий

*   **Фреймворк:** Nuxt 3 (Vue 3, Composition API)
*   **Язык:** TypeScript
*   **Стилизация:** Vanilla CSS + Bootstrap Grid (адаптивная сетка)
*   **База данных / API:** Firebase Realtime Database (REST API)
*   **Сборщик / Инструменты:** Vite + gh-pages для автодеплоя

## 🚀 Быстрый старт

### 1. Установка зависимостей
Убедитесь, что у вас установлен Node.js (рекомендуется v18+). Установите библиотеки проекта:
```bash
npm install
```

### 2. Запуск сервера разработки
Запустите локальный сервер для отладки приложения на `http://localhost:3000`:
```bash
npm run start
```

### 3. Сборка для продакшена
Скомпилируйте проект под серверное окружение:
```bash
npm run build
```

## 📦 Деплой на GitHub Pages

Данный проект полностью адаптирован под статический хостинг. 

### 1. Настройка базового адреса
В файле [`nuxt.config.ts`](file:///c:/Users/gnz29/Documents/dev/2026/Nuxt/wb-nuxt/nuxt.config.ts) измените параметр `app.baseURL` на имя вашего репозитория:
```typescript
app: {
  baseURL: '/имя-вашего-репозитория/',
}
```

### 2. Сборка статических файлов
Сгенерируйте статические страницы проекта (все данные загрузятся в `.output/public`):
```bash
npm run generate
```

### 3. Публикация
Отправьте собранный проект в ветку `gh-pages` одной командой:
```bash
npm run deploy
```

После этого перейдите в настройки репозитория на GitHub (**Settings -> Pages**) и укажите ветку `gh-pages` в качестве источника для деплоя.
